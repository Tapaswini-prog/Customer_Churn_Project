# 📊 Customer Churn Prediction

A Machine Learning project to predict whether a customer will leave a telecom company (churn) using Python and Scikit-learn.

---

## 🚀 Project Overview

Customer churn prediction is a crucial problem for businesses. This project aims to identify customers who are likely to leave based on historical data.

By predicting churn early, companies can take actions to retain customers and reduce losses.

---

## 🧠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib (optional if used)
